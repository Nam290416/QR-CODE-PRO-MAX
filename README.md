# QR CODE PRO MAX

QR Generator viết bằng Python.

## Tính năng

- Tạo QR Code
- Chỉnh màu QR
- RGB / HSL
- HEX Color
- Lưu PNG
- In QR
- Đọc QR từ ảnh
- Scan QR bằng Webcam
- Khung viền QR

## Cài đặt

```bash
pip install -r requirements.txt
```

## Chạy

```bash
python QR_CODE_PRO_MAX.py
```
